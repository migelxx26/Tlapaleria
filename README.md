## Deploy en GitHub Pages
1. Crea un repositorio y sube estos archivos a la rama `main`.
2. Ve a **Settings → Pages** y elige **Deploy from branch** (main / root).
3. Tu sitio quedará en `https://<tu-usuario>.github.io/<repo>/`.

> Si cambias el dominio, actualiza `og:url`, `og:image` y `sitemap.xml`.
